## 技术栈

### 机械
- 铝型材: 4040/4080欧标
- 丝杆: 1610(XY) / 1605(Z)
- 导轨: MGN12
- 电机: 57闭环(XY) / 57+减速(Z1) / 42(Z2+E)

### 光学
- 激光: 200W光纤 1064nm
- 准直镜: F50
- 聚焦镜: F100
- 光斑: ~100μm

### 控制
- 主控: MKS Robin Nano V3 (STM32F407)
- 固件: Marlin (需修改)
- 通信: USB/串口

### 气体
- 保护气: 氩气 40L
- 目标氧含量: <500ppm
- 监测: 电化学氧分析仪
