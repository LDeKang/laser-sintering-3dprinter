## 工具调用策略

### 优先使用
1. `read` - 读取项目文档
2. `write` - 创建/更新文件
3. `exec` - 执行构建/测试命令
4. `web_search` - 查询技术资料

### 谨慎使用
- `edit` - 必须精确匹配，先read再edit
- `browser` - 只在需要交互时使用
- `message` - 确认用户需要发送才用

### 禁止使用 (本项目)
- 无外部API调用权限
- 无硬件控制权限
- 无自动采购权限

## 决策逻辑

```
IF 用户问技术问题:
    IF 在MEMORY.md中有记录 → 直接引用
    ELSE → 搜索/推理 → 记录到MEMORY.md
    
IF 用户要修改方案:
    IF 影响预算 > 10% → 明确提示风险
    IF 影响安全 → 强制确认
    ELSE → 执行并记录变更
    
IF 检测到错误:
    立即承认 → 分析根因 → 给出修复 → 记录到REFLECTION.md
```

## 项目特定工作流

### SLM打印机项目

**Phase 1: 机械装配**
- 每日检查: 零件到货状态
- 里程碑: 框架搭建完成
- 输出: 装配照片/视频记录

**Phase 2: 光路搭建**
- 安全检查: 激光器功率测试
- 里程碑: 焦点对准
- 输出: 光路对准报告

**Phase 3: 控制系统**
- 代码审查: Marlin修改
- 里程碑: 各轴运动正常
- 输出: 固件版本+配置

**Phase 4: 工艺调试**
- 数据记录: 每次烧结参数
- 里程碑: 10mm立方体
- 输出: 工艺参数表
