# SLM金属3D打印机详细设计规格书 v1.0

## 1. 项目概述

### 1.1 项目目标
构建一台基于机械XY点烧结方案的入门级SLM（选择性激光熔化）金属3D打印机，预算¥15000，能够打印316L不锈钢零件。

### 1.2 技术路线选择
| 方案 | 优点 | 缺点 | 选择 |
|------|------|------|------|
| 振镜扫描 | 速度快(1-2m/s)、精度高 | 成本高(¥3500+)、复杂 | ❌ |
| 机械XY点烧结 | 成本低、结构简单 | 极慢(50mm/s)、精度一般 | ✅ |

### 1.3 核心参数

| 参数 | 规格 | 备注 |
|------|------|------|
| 成型尺寸 | 100×100×80mm | X×Y×Z |
| 激光功率 | 200W | 光纤激光器 1064nm |
| 光斑直径 | ~100μm | 聚焦后 |
| 层厚范围 | 50-100μm | 可调 |
| XY分辨率 | 10μm | 步进电机+滚珠丝杆 |
| Z轴分辨率 | 2.5μm | 减速后 |
| 扫描速度 | 50mm/s | 点烧结模式 |
| 单层时间 | 5-10分钟 | 取决于填充率 |
| 材料 | 316L不锈钢 | 球形粉末 15-45μm |

## 2. 机械系统设计

### 2.1 整体布局

```
      [激光器]
         │
         ▼ (光纤)
    [准直聚焦镜组]
         │
         ▼
    ╔═══════════════╗
    ║  [保护舱]      ║
    ║  ┌─────────┐  ║
    ║  │ XY扫描  │  ║ ← X轴(横向) + Y轴(纵向)
    ║  │   激光头 │  ║
    ║  └─────────┘  ║
    ║       │       ║
    ║       ▼       ║
    ║  ┌─────────┐  ║
    ║  │ 成型平台│  ║ ← Z1轴(下降)
    ║  └─────────┘  ║
    ║  ┌─────────┐  ║
    ║  │ 送粉缸  │  ║ ← Z2轴(上升)
    ║  └─────────┘  ║
    ╚═══════════════╝
```

### 2.2 X轴（激光头横向）

**功能**：携带激光头横向移动，实现X方向扫描

| 参数 | 规格 |
|------|------|
| 行程 | 120mm |
| 驱动 | 57闭环步进电机 |
| 减速比 | 1:5 |
| 传动 | 1610滚珠丝杆（导程10mm）|
| 导轨 | MGN12直线导轨×2 |
| 分辨率 | 10μm/步 |
| 最大速度 | 50mm/s |
| 定位精度 | ±0.05mm |

**选型理由**：
- 57电机+1:5减速：提供足够扭矩，闭环防止丢步
- 1610丝杆：10mm导程平衡速度和精度
- MGN12导轨：承载激光头（约2kg）稳定运行

### 2.3 Y轴（激光头纵向）

**功能**：携带X轴组件纵向移动，实现Y方向扫描

| 参数 | 规格 |
|------|------|
| 行程 | 120mm |
| 驱动 | 57闭环步进电机 |
| 减速比 | 1:5 |
| 传动 | 1610滚珠丝杆（导程10mm）|
| 导轨 | MGN12直线导轨×2 |
| 负载 | X轴组件+激光头（约5kg）|

**注意**：Y轴负载较大，需要更强的电机或更大减速比

### 2.4 Z1轴（成型缸）

**功能**：成型平台逐层下降

| 参数 | 规格 |
|------|------|
| 行程 | 100mm |
| 驱动 | 57步进电机 |
| 减速比 | 1:10行星减速器 |
| 传动 | 1605滚珠丝杆（导程5mm）|
| 分辨率 | 2.5μm/步 |
| 负载 | 平台+粉末（约5kg）|

**选型理由**：
- 1:10减速：提供大扭矩支撑粉末重量
- 1605丝杆：5mm导程提高分辨率
- 高精度要求：层厚控制关键

### 2.5 Z2轴（送粉缸）

**功能**：送粉平台逐层上升，补充粉末

| 参数 | 规格 |
|------|------|
| 行程 | 100mm |
| 驱动 | 42步进电机 |
| 传动 | 1605滚珠丝杆（导程5mm）|
| 分辨率 | 25μm/步（无减速）|

**注意**：送粉精度要求较低，42电机足够

### 2.6 铺粉系统

**功能**：将粉末从送粉缸推到成型区域并刮平

| 参数 | 规格 |
|------|------|
| 驱动 | 42步进电机 |
| 传动 | GT2同步带，20齿皮带轮 |
| 行程 | 150mm |
| 刮刀 | 硬质合金，厚度0.3mm |
| 安装 | 磁吸快换 |
| 铺粉速度 | 100mm/s |

**工作流程**：
1. 刮刀在送粉缸上方待命
2. Z2上升一定量（送粉）
3. 刮刀移动，将粉末推到成型区域
4. 刮刀返回原位
5. Z1下降一层厚度

## 3. 激光系统

### 3.1 激光器

| 参数 | 规格 |
|------|------|
| 类型 | 光纤激光器 |
| 功率 | 200W |
| 波长 | 1064nm |
| 光束质量 | M² < 1.5 |
| 调制方式 | TTL/PWM |
| 调制频率 | 1-50kHz |
| 冷却 | 风冷/水冷 |
| 光纤芯径 | 20μm |

**采购建议**：
- 国产锐科/创鑫二手设备
- 要求带保修（至少3个月）
- 确认光纤接口类型（QBH或QCS）

### 3.2 光路系统

```
激光器光纤输出
    │
    ▼
[准直镜] ──► 平行光束（直径约10mm）
    │
    ▼
[聚焦镜] ──► 聚焦光斑（直径约100μm）
    │
    ▼
[保护镜片] ──► 防止飞溅污染
    │
    ▼
   工件表面
```

| 部件 | 规格 |
|------|------|
| 准直镜 | F=50mm |
| 聚焦镜 | F=100mm |
| 工作距离 | 约80mm |
| 保护镜片 | 石英玻璃，镀1064nm增透膜 |

**光斑计算**：
- 光纤芯径20μm，NA≈0.1
- 准直后光束直径≈10mm
- 聚焦后光斑直径≈100μm

### 3.3 激光控制

| 控制方式 | 信号类型 | 说明 |
|----------|----------|------|
| 使能 | TTL高电平 | 激光器准备就绪 |
| 调制 | PWM | 功率调节 0-100% |
| 急停 | 开关量 | 安全联锁 |

**PWM参数**：
- 频率：1-10kHz（机械扫描慢，不需要高频）
- 占空比：0-100%对应功率0-200W
- 建议工作功率：100-150W（留余量）

## 4. 保护舱设计

### 4.1 功能要求

1. **惰性气氛**：氧含量<500ppm，防止金属氧化
2. **安全防护**：防止激光泄漏、粉末飞溅
3. **观察窗口**：便于监控打印过程
4. **操作接口**：手套口或快开门

### 4.2 结构设计

```
        ┌─────────────────┐
       ╱    [观察窗]       ╲
      │    ┌─────────┐     │
      │    │  照明灯  │     │
      │    └─────────┘     │
      │  ╔═══════════════╗  │
      │  ║               ║  │
[手套口]║   打印区域     ║[手套口]
      │  ║               ║  │
      │  ╚═══════════════╝  │
      │    [进气口] [出气口]  │
       ╲                   ╱
        └─────────────────┘
              [门]
```

### 4.3 材料与加工

| 部件 | 材料 | 厚度 | 加工方式 |
|------|------|------|----------|
| 箱体 | 304不锈钢 | 2mm | 焊接 |
| 框架 | 4040铝型材 | - | 组装 |
| 观察窗 | 激光防护玻璃 | 5mm | 镶嵌 |
| 手套口 | 标准手套法兰 | - | 安装 |
| 密封条 | 硅胶条 | - | 粘贴 |

### 4.4 气体系统

**气体流程**：
```
氩气瓶 → 减压阀 → 流量计 → 保护舱底部进气
                              ↓
                    [氧含量传感器监测]
                              ↓
                         顶部出气 → 排出或回收
```

| 部件 | 规格 |
|------|------|
| 氩气瓶 | 40L工业瓶 |
| 减压阀 | 0-0.6MPa可调 |
| 流量计 | 0-50L/min |
| 氧分析仪 | 0-1000ppm，电化学传感器 |

**操作参数**：
- 充气流量：10-20L/min
- 目标氧含量：<500ppm
- 换气次数：建议5-10次舱体积

## 5. 控制系统

### 5.1 控制架构

```
PC (上位机 - 切片+控制软件)
    │ USB/串口
    ▼
[MKS Robin Nano V3] (主控板)
    ├─ X轴驱动 ──► 57闭环步进
    ├─ Y轴驱动 ──► 57闭环步进
    ├─ Z1驱动 ───► 57步进(成型缸)
    ├─ Z2驱动 ───► 42步进(送粉缸)
    ├─ E轴驱动 ──► 42步进(刮刀)
    ├─ PWM输出 ──► 激光器调制
    ├─ TTL输出 ──► 激光器使能
    └─ 输入接口 ◄── 限位开关/急停/门开关
```

### 5.2 主控板选型

**MKS Robin Nano V3 规格**：
- MCU: STM32F407VGT6 (168MHz)
- 步进驱动: 5路 (TMC2209或A4988)
- 输入: 限位开关×6，温度×2，其他
- 输出: PWM×2，风扇×2，加热×2
- 接口: USB, TF卡, WiFi(可选)

**选型理由**：
- 32位MCU，性能足够
- 5路步进，刚好满足需求
- 开源Marlin固件支持
- 性价比高 (~¥300)

### 5.3 安全联锁

| 联锁项目 | 传感器 | 动作 |
|----------|--------|------|
| 舱门开启 | 门磁开关 | 激光器关闭 |
| 急停按钮 | 急停开关 | 全系统断电 |
| 氧含量过高 | 氧分析仪 | 报警+暂停打印 |
| 激光器故障 | 激光器反馈 | 暂停打印 |

## 6. 软件系统

### 6.1 固件修改 (基于Marlin)

**需要修改的内容**：

1. **激光控制指令**
   - M3 Sxxx: 设置激光功率 (0-255对应0-200W)
   - M5: 关闭激光
   - M3 I: 激光器使能

2. **点烧结模式**
   - G1 Xx Yy F50: 移动到位置 (速度50mm/s)
   - G4 Pxx: 停留时间 (ms)
   - M3 Sxxx: 开激光
   - G4 Pxx: 烧结时间
   - M5: 关激光

3. **自动铺粉流程**
   - 自定义G代码宏实现铺粉动作

4. **层间处理**
   - 每层结束后自动执行铺粉

### 6.2 切片软件

**方案**：自研简化切片器

**输入**：STL文件 + 工艺参数
**输出**：自定义G代码

**切片流程**：
1. 读取STL，三角面片化
2. 分层（50-100μm层厚）
3. 每层生成填充路径（蛇形扫描）
4. 转换为点烧结G代码

### 6.3 上位机软件

**功能**：
- 文件管理（STL导入、G代码保存）
- 参数设置（功率、速度、层厚等）
- 打印控制（开始、暂停、停止）
- 实时监控（温度、氧含量、进度）
- 手动控制（各轴移动、激光测试）

## 7. 工艺参数

### 7.1 316L不锈钢推荐参数

| 参数 | 初始值 | 范围 | 说明 |
|------|--------|------|------|
| 激光功率 | 120W | 80-150W | 根据扫描速度调整 |
| 扫描速度 | 50mm/s | 30-80mm/s | 机械限制 |
| 光斑直径 | 100μm | - | 光学系统决定 |
| 层厚 | 50μm | 30-100μm | 影响表面质量 |
| 扫描间距 | 80μm | 60-100μm | 搭接率20-40% |
| 烧结时间 | 2ms | 1-5ms | 点烧结停留时间 |

### 7.2 工艺优化方向

1. **提高致密度**：
   - 增加激光功率
   - 减小层厚
   - 优化扫描策略（棋盘格、岛式）

2. **改善表面质量**：
   - 减小层厚
   - 减小扫描间距
   - 优化铺粉质量

3. **减少变形**：
   - 添加支撑结构
   - 预热基板
   - 优化扫描顺序

## 8. 风险与应对

| 风险 | 概率 | 影响 | 应对措施 |
|------|------|------|----------|
| 激光器故障 | 低 | 高 | 买带保修的二手，预留维修预算 |
| 机械精度不足 | 中 | 中 | 闭环步进+定期校准 |
| 氧含量控制不好 | 中 | 中 | 先打不锈钢，活性金属延后 |
| 粉末流动性差 | 中 | 中 | 选球形316L粉末，控制湿度 |
| 层间结合不良 | 中 | 高 | 工艺参数优化，提高功率 |
| 打印件变形开裂 | 中 | 中 | 添加支撑，控制热应力 |
