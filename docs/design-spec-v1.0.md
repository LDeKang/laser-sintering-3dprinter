# 激光烧结 3D 打印机改装方案

**项目**: 3D打印机改装选择性激光烧结(SLS)设备  
**版本**: v1.0  
**日期**: 2026-03-14  
**周期**: 1个月

---

## 1. 项目概述

### 1.1 目标
将普通 FDM 3D 打印机改装为桌面级选择性激光烧结(SLS)设备，实现尼龙粉末的逐层激光烧结成型。

### 1.2 核心参数

| 参数 | 规格 |
|---|---|
| 成型尺寸 | 200×200×150mm（待定） |
| 激光功率 | 10W 450nm 半导体激光 |
| 激光头重量 | 1kg |
| 铺料机构重量 | 1.5kg |
| 单次铺料量 | 30g |
| 层厚 | 0.1-0.2mm |
| 扫描速度 | 10-50mm/s |
| 控制主板 | BTT SKR Mini E3 V3.0 (STM32) |
| 固件 | Marlin 2.1.2.1 |

## 2. 系统架构

### 2.1 机械结构

```
┌─────────────────────────────────────────┐
│  [激光头模块]  ← X轴电机 (左右移动)       │
│     重量: 1kg                            │
│     功率: 10W 450nm                     │
│     PWM控制: FAN0接口                    │
└─────────────────────────────────────────┘
                   ↓ 激光束
┌─────────────────────────────────────────┐
│  [成型腔体]                               │
│  ┌─────────────────────────────────┐    │
│  │      烧结区域 (200×200mm)        │    │ ← XY扫描范围
│  │                                 │    │
│  │    ████████████████████████    │    │ ← 粉末床
│  │    ████ 烧结层 (0.1mm) ████    │    │
│  └─────────────────────────────────┘    │
│           ↓ Z轴下降                      │
│  ┌─────────────────────────────────┐    │
│  │      成型平台 (Z轴)              │    │ ← 步进电机驱动
│  │      行程: 150mm                │    │
│  └─────────────────────────────────┘    │
└─────────────────────────────────────────┘
                   ↓
┌─────────────────────────────────────────┐
│  [铺料系统]                               │
│  ┌─────────────────────────────────┐    │
│  │  [刮板] ← 铺料电机 (E0轴)        │    │ ← 步进电机
│  │   行程: 220mm                   │    │
│  │   速度: 20mm/s                  │    │
│  └─────────────────────────────────┘    │
│  ┌─────────────────────────────────┐    │
│  │      粉料存储区 (30g)            │    │
│  └─────────────────────────────────┘    │
└─────────────────────────────────────────┘
```

### 2.2 运动轴分配

| 轴 | 功能 | 电机类型 | 备注 |
|---|---|---|---|
| **X** | 激光头左右移动 | 42步进电机 | 原X轴 |
| **Y** | 激光头前后移动 | 42步进电机 | 原Y轴 |
| **Z** | 成型平台下降 | 42步进电机 | 原Z轴 |
| **E0** | 铺料刮板往复 | 42步进电机 | 原挤出机改 |

## 4. 固件配置方案

### 4.1 Marlin 关键配置

#### Configuration.h

```cpp
// 主板选择
#define MOTHERBOARD BOARD_BTT_SKR_MINI_E3_V3_0

// 打印机名称
#define CUSTOM_MACHINE_NAME "Laser Sintering"

// XYZ 行程 (根据实际机器调整)
#define X_BED_SIZE 200
#define Y_BED_SIZE 200
#define Z_MAX_POS 150

// 禁用热端 (激光不需要)
#define EXTRUDERS 0
#define TEMP_SENSOR_0 0

// 启用热床 (平台预热)
#define TEMP_SENSOR_BED 1
#define BED_MAXTEMP 180  // 尼龙预热温度
```

#### Configuration_adv.h

```cpp
// 启用激光功能
#define LASER_FEATURE

// PWM 配置
#define SPINDLE_LASER_USE_PWM
#define SPINDLE_LASER_FREQUENCY 1000  // 1kHz
#define SPINDLE_LASER_PWM_INVERT false
#define SPINDLE_LASER_ACTIVE_STATE LOW

// 功率单位
#define CUTTER_POWER_UNIT PWM255  // 0-255

// 激光移动时自动开启
#define LASER_MOVE_POWER
```

### 4.2 铺料电机配置

```cpp
// E0 轴改为线性轴 (非挤出机)
#define DISTINCT_E_FACTORS
#define DEFAULT_AXIS_STEPS_PER_UNIT { 80, 80, 400, 80 }  // E轴改为铺料

// 铺料电机参数
#define DEFAULT_MAX_FEEDRATE { 300, 300, 5, 20 }  // E轴速度限制
#define DEFAULT_MAX_ACCELERATION { 3000, 3000, 100, 500 }
```

### 4.3 G-code 指令映射

| 指令 | 功能 |
|---|---|
| `M3 S255` | 激光最大功率 |
| `M3 S128` | 50% 功率 |
| `M5` | 关闭激光 |
| `G1 E50 F1000` | 铺料前进 |
| `G1 E0 F3000` | 铺料复位 |
| `M140 S160` | 平台预热 160°C |

## 5. 软件流程

### 5.1 切片流程

```
3D模型 (STL)
    │
    ▼
PrusaSlicer
- 层高: 0.1mm
- 填充: 100% (实心)
- 速度: 20mm/s
- 填充角度: 45°/135°交替
    │
    ▼
G-code后处理脚本
- 替换 E→激光功率
- 添加铺料指令
- 添加层间延迟
    │
    ▼
最终G-code → 打印机
```

### 5.2 单层G-code示例

```gcode
; 第N层开始
G1 Z0.1 F300        ; Z轴下降
M3 S200             ; 开启激光 78%功率
G1 X10 Y10 F600     ; 移动至起点
G1 X50 Y10          ; 扫描线1
G1 X50 Y12          ; 偏移
G1 X10 Y12          ; 扫描线2
...                 ; 重复扫描
M5                  ; 关闭激光
G1 E50 F1000        ; 铺料前进
G4 P500             ; 等待500ms
G1 E0 F3000         ; 铺料复位
; 第N层结束
```

## 7. 风险分析

### 7.1 技术风险

| 风险 | 概率 | 影响 | 应对措施 |
|---|---|---|---|
| 激光功率不足 | 中 | 高 | 备用20W激光头 |
| XY轴负载过重 | 中 | 高 | 降速/加固结构 |
| 铺粉不均匀 | 高 | 中 | 手动辅助/调整刮板 |
| 粉末粘连不良 | 中 | 高 | 平台预热/降低速度 |
| 翘边变形 | 中 | 中 | 优化扫描路径 |

### 7.2 安全风险

| 风险 | 等级 | 防护措施 |
|---|---|---|
| 激光伤害 | 高 | 防护眼镜/密封罩 |
| 粉末易燃 | 中 | 惰性气体/通风 |
| 高温烫伤 | 中 | 警示标识/隔热 |
