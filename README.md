# SLM Metal 3D Printer

选择性激光熔化金属3D打印机 - Selective Laser Melting Metal 3D Printer

## 项目概述

**机械XY点烧结方案** - 极慢但能做出来

将机械XY运动系统与光纤激光器结合，实现入门级SLM金属3D打印。

### 核心参数

| 参数 | 值 |
|---|---|
| 技术路线 | SLM (选择性激光熔化) |
| 材料 | 316L不锈钢粉末 |
| 激光器 | 200W 光纤激光器 1064nm |
| 扫描方式 | 机械XY点烧结 (无振镜) |
| 成型尺寸 | 100×100×80mm |
| 层厚 | 50-100μm |
| 光斑直径 | ~100μm |
| 扫描速度 | 50mm/s (点烧结模式) |
| 单层时间 | 5-10分钟 |
| 预算 | ¥15000 |

### 运动方案

- **X轴**: 57闭环步进 + 1610滚珠丝杆，激光头横向移动
- **Y轴**: 57闭环步进 + 1610滚珠丝杆，激光头纵向移动
- **Z1轴**: 57步进+1:10减速 + 1605滚珠丝杆，成型缸下降
- **Z2轴**: 42步进 + 1605滚珠丝杆，送粉缸上升
- **E轴**: 42步进 + 同步带，刮刀铺粉

## 关键妥协与风险

### 已接受的妥协
1. **速度极慢**: 点烧结模式，单层5-10分钟
2. **精度一般**: 机械间隙+步进分辨率限制
3. **表面粗糙**: Ra 15-25μm，需后处理
4. **材料受限**: 只能打316L不锈钢

### 主要风险
| 风险 | 概率 | 影响 | 应对 |
|---|---|---|---|
| 激光器功率衰减 | 低 | 高 | 买带保修的二手 |
| 氧含量控制不好 | 中 | 中 | 先打不锈钢，活性金属延后 |
| 粉末流动性差 | 中 | 中 | 选球形316L粉末，粒径15-45μm |
| 机械精度不够 | 中 | 中 | 闭环步进+定期校准 |
